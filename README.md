# SchoolGradesSystem — Веб‑система учёта школьных предметов и анализа оценок

Учебный веб‑проект, созданный в **Eclipse**, с использованием **Java**, **JSP**, **Servlet API**, **SQLite** и **Apache Tomcat**.  
Система позволяет вести учёт учеников, предметов, оценок и автоматически рассчитывать средние показатели успеваемости.

## Функциональные возможности

## Запуск проекта

### Требования
- Eclipse IDE (с поддержкой Dynamic Web Project)
- Apache Tomcat 9 (подключён в Eclipse)
- Java JDK 8+  
- JDBC‑драйвер sqlite-jdbc (добавлен в lib проекта)

---

### 1. Импорт проекта в Eclipse
File → Import → Existing Project → указать папку проекта.

### 2. Настройка Tomcat
Window → Preferences → Server → Runtime Environments → Add → Apache Tomcat 9.

### 3. Запуск
Правый клик по проекту → **Run on Server**  
После запуска приложение доступно по адресу:

```
http://localhost:8080/SchoolGradesSystem_2/index.jsp
```
